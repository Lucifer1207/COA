RAM

(RANDOM ACCESS MEMORY)

****Random-access memory is a form of computer memory that can be read and changed in any
order,typically used to store working data and machine code.A random-access memory device
allows data items to be read or written in almost the same amount of time irrespective of
the physical location of data inside the memory.****

****In today's technology,random-access memory takes the form of integrated circuit(IC)chips with MOS
(metal–oxide–semiconductor) memory cells.RAM is normally associated with volatile types of memory where
stored information is lost if power is removed.****

TYPES OF RAM



****There Are Mainly Two Types Of Ram****
1.Static-Random Access Memory(SRAM)
2.Dynamic-Random Access Memory(DRAM)



1.Static-Random Access Memory(SRAM)

*It Store The Binary Information In Flip-Flop.
*Fewer Memory Capacities.
*It Has Faster Access Time.
*Used In Implementing Cache Memory.
*Refreshing Is Not Needed To Keep Data Intact.
*High Cost Of Manufacturing.
*Low Power Consumption.

2.Dynamic-Random Access Memory(DRAM)

*It Store The Binary Information As The Charge On the Capacitor.
*Greater Memory Capacities.
*It Has slow Access speed.
*Used In Implementing Main Memory.
*Refreshing Is Needed To Keep Data Intact.
*Low Cost Of Manufacturing.
*High Power Consumption.
