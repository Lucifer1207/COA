RISC & CISC

RISC

*Reduced Instruction Set Computer
*Highly Pipelined
*Simple Instructions Are Few In Numbers
*3 Register Operand Allowed Per Instruction(add R1,R2,R3)
*Few Addressing Modes
*Complexity In Compiler
*Hardwired Control Unit
*Multiple Register Set(256)
*Single-Cycle Instructions(except load & store)
*Fixed Lenght Instructions

CISC

*Complex Instruction Set Computer
*Less Pipelined
*Complex Instructions Are Many In Number
*1 or 2 Register Operand Allowed Per Instruction(add R1,R2,)
*Many Addressing Modes
*complexity In Microcode
*Micro-Programmed Control Unit
*Single Register Set(6-16)
*Multiple-Cycle Instructions
*Variable length Instruction
