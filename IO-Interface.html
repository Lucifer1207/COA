I/O INTERFACE

****Input-Output Interface is used as an method which helps in transferring of information
between the internal storage devices i.e. memory and the external peripheral device.
A peripheral device is that which provide input and output for the computer,it is also called Input-Output Devices.
For Example: A keyboard and mouse provide Input to the computer are called input devices while a monitor and printer
that provide output to the computer are called output devices. Just like the external hard-drives, there is also
availability of some peripheral devices which are able to provide both input and output.****

TYPES OF I/O

****There Are Mainly Three Types Of I/O****
1.On The Basis Of Memory.
2.On The Basis Of Transmission.
3.On The Basis Of Synchronization.

1.On The Basis Of Memory

A)ISOLATED.
B)MEMORY MAPPED.

A)Isolated..

*Isolated I/O Uses Separate Memory Space.
*Slow Operation.
*Larger In Size Due To More Buses.
*More Efficient Due To Separate Buses.
*Uses Complex Internal Logic.
*The Addresses For Isolated I/O Devices Are Called Ports.
*Limited Instructions Can Be Used.

B)Memory Mapped..

*Memory mapped I/O uses memory from the main memory.
*Fast Operation.
*Smaller In Size Due To Less Buses.
*Inefficient I/O operations due to using single bus for data and addressing.
*Common internal logic for memory and I/O devices.
*Memory Mapped I/O Devices Are Treated As Memory Locations On The Memory Map.
*Any Instruction Which References To Memory Can Be Used.

1.On The Basis Of Transmission

A)SERIAL TRANSMISSION.
B)PARALLEL TRANSMISSION.

A)Serial Transmission..

*Shift Register Is Used.
*Burst Error Takes Place.
*Uses For Long Distance Transmission.
*Transmission Is Slow.
*Economical.
*Simple And Reliable.
*Full Duplex.
*1 Bit Transfered At 1 Clock Pulse.
*Converters Are Used.
*Ex-KEYBOARD.

A)Parallel Transmission..

*Shift Register Is Not Used.
*Bit Error Takes Place.
*Uses For Short Distance Transmission.
*Transmission Is Fast.
*Expensive.
*Complex And Unreliable.
*Half Duplex.
*8 Bits(1byte) Transfered at 1 clock pulse.
*Converters Are Not Required.
*Ex-PRINTER.

3.On The Basis Of Synchronization

A)SYNCHRONOUS.
B)ASYNCHRONOUS.

A)Synchronous..

*Transmission starts with the block header which holds a sequence of bits.
*Implemented By Hardware And Software.
*There Exist No Gap b/w The Data.
*Efficient.
*Transmission Is Fast.
*Expensive.
*Reliable.
*Full Duplex.
*Data Sent In Form Of Blocks/Frames.
*Need Synchronization.
*Ex-Chat Rooms,Telephonic Communication,Video Conferencing.

B)Aynchronous..

*It uses start bit and stop bit preceding and following a character respectively.
*Implemented By Hardware Only
*There Exist Gap b/w The Data.
*InEfficient.
*Transmission Is Slow.
*Economical.
*UnReliable.
*Half Duplex.
*Data Sent In Form Of Byte.
*No Need Of Synchronization.
*Ex-Emails,Letters,Forums.
