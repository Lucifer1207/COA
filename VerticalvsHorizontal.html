HORIZONTAL µ-PROGRAMMED CU &
VERTICAL µ-PROGRAMMED CU....

HORIZONTAL µ-PROGRAMMED

*t supports longer control word.
*It allows a higher degree of parallelism.
*No additional hardware is required.
*It is faster than a Vertical micro-programmed control unit.
*It is more flexible than a vertical micro-programmed control unit.
* makes less use of ROM encoding than the vertical micro-programmed control unit.
*uses horizontal micro-instruction
* every bit in the control field attaches to a control line.

VERTICAL µ-PROGRAMMED

*It supports shorter control word.
*It allows a low degree of parallelism.
*Additional hardware in the form of decoders is required to generate control signals.
*it is slower than a Horizontal micro-programmed control unit.
*It is less flexible than horizontal but more flexible than that of a hardwired control unit.
*makes more use of ROM encoding to reduce the length of the control word.
*uses vertical micro-instruction
*a code is used for each action to be performed and the decoder translates this code into individual control signals.
